## 🔧 사용 기술

- **Frontend:** Next.js 15, TypeScript
- **Styling:** Tailwind CSS
- **API 통신:** Fetch API
- **배포:** Vercel

---


## 🗂 페이지별 주요 기능

### 홈 페이지
- 전체 할 일 목록 조회
- 완료 여부 체크
- 할 일 생성 폼
- 반응형 레이아웃

### 할 일 상세 페이지
- 할 일 이름, 메모, 이미지 확인
- 할 일 수정 및 이미지 업로드
- 삭제 기능
- 완료 상태 토글
